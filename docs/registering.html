(un)wrapping plotly figures#

Register plotly-resampler to (un)wrap plotly-graph-objects.

plotly_resampler.registering.register_plotly_resampler(mode='auto', **aggregator_kwargs)[source]#

Register plotly-resampler to plotly.graph_objects.

This function results in the use of plotly-resampler under the hood.

Note

We advise to use mode= widget when working in an IPython based environment as this will just behave as a go.FigureWidget, but with dynamic aggregation. When using mode= auto or figure; most figures will be wrapped as FigureResampler, on which show_dash needs to be called.

If ‚Äòfigure‚Äô is used, all plotly figures are wrapped as FigureResampler objects. If ‚Äòwidget‚Äô is used, all plotly figure widgets are wrapped as FigureWidgetResampler objects (we advise to use this mode in IPython environment with a kernel). If None is used, wrapping is done as expected (go.Figure -> FigureResampler, go.FigureWidget -> FigureWidgetResampler).

  • aggregator_kwargs (dict, optional) ‚Äì The keyword arguments to pass to the plotly-resampler decorator its constructor. See more details in FigureResampler and FigureWidgetResampler.
